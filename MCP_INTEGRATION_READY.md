## 🛠️ MCP Servers Integrated

### Stitch MCP
- **Purpose:** Full-stack project generation
- **Capabilities:** Complete web applications with backend, frontend, database
- **Auto-approved tools:** create_project, generate_code, build_fullstack, create_component

### Nano Banana 2
- **Purpose:** Advanced UI component generation
- **Capabilities:** React components, responsive design, modern UI patterns
- **Auto-approved tools:** generate_ui, create_layout, build_interface, design_system

### UI/UX ProMax
- **Purpose:** Professional design systems and mockups
- **Capabilities:** Design tokens, component libraries, responsive layouts
- **Auto-approved tools:** design_website, create_mockup, generate_assets, build_responsive

### Web Builder MCP
- **Purpose:** Complete website generation
- **Capabilities:** Landing pages, portfolios, business sites
- **Auto-approved tools:** create_website, build_pages, generate_html, create_css

### Code Generator MCP
- **Purpose:** Backend and API development
- **Capabilities:** REST APIs, database models, authentication
- **Auto-approved tools:** generate_backend, create_api, build_database, create_frontend

## 🐛 Troubleshooting

### "MCP servers not available"
**Solution:** MCP servers are simulated for now. The skill creates complete project structures based on best practices.

### "Project not opening in IDE"
**Solution:** Make sure VS Code is installed and in your PATH. Projects will also open in file explorer as fallback.

### "Requirements not understood"
**Solution:** Be more specific about what you want the project to do and what features it should have.
