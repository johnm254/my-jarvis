## 💡 Pro Tips

### 1. YouTube Auto-Play
The keyboard method (Tab + Enter) is most reliable and works on all screens!

### 2. Typing
Make sure the target window is focused. Say "Click" first if needed.

### 3. Natural Language
Use natural commands:
- "Make it louder" = Volume up
- "Find my document" = Search for document
- "Close this" = Close window

### 4. Wait for Actions
Give JARVIS a moment between commands for actions to complete.

### 5. Chain Commands
You can give multiple commands in sequence!

## 🐛 If YouTube Doesn't Play

### Solution 1: Increase Wait Time
Edit `jarvis/skills/music_player.py` line ~75:
```python
time.sleep(6)  # Change to 8 or 10 for slow internet
```

### Solution 2: Adjust Click Positions
Add your screen-specific position in the `possible_positions` list.

### Solution 3: Use Keyboard Method
The keyboard fallback (Tab 15x + Enter) works automatically and is most reliable!
