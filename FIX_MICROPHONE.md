# 🎤 Fix Microphone Issue - Quick Guide

## ⚠️ Issue
```
Warning: Microphone not available: Could not find PyAudio
```

---

## ✅ SOLUTION 1: Use Text Mode (Recommended - Works Now!)

### Start Text Mode
```bash
python jarvis_text.py
```

**All features work - just type instead of speak!**

### Example
```

[YouTube opens and plays automatically]

🗣️  JARVIS: Volume increased to 60%

👤 You: Type hello world
🗣️  JARVIS: Typed: hello world
```

**Everything works perfectly in text mode!**

---

## ✅ SOLUTION 2: Install PyAudio for Voice Mode

### Quick Install (5 minutes)

**Step 1: Install C++ Build Tools**
1. Download: https://visualstudio.microsoft.com/visual-cpp-build-tools/
2. Run installer
3. Select "Desktop development with C++"
4. Click Install
5. Restart computer

**Step 2: Install PyAudio**
```bash
pip install pyaudio
```

**Step 3: Test Voice Mode**
```bash
python conversational_jarvis.py
```

Now speak your commands!

## 🎤 Voice Mode Setup (Optional)

### For Python 3.14 (Your Version)

PyAudio may not have pre-built wheels for Python 3.14 yet.

**Option A: Install C++ Build Tools**
1. Download Visual Studio Build Tools
2. Install "Desktop development with C++"
3. Run: `pip install pyaudio`

**Option B: Use Python 3.11**
1. Install Python 3.11
2. Create virtual environment
3. Install dependencies
4. PyAudio will work

**Option C: Wait for PyAudio Update**
- PyAudio will eventually support Python 3.14
- Use text mode until then
